We commit to developing and openly releasing all relevant software as free software, adhering to modern best practices.

Most of our software stack is built upon the Python programming language and the
associated scientific ecosystem.


## Links

1. [aptapy](https://github.com/lucabaldini/aptapy):
   a set of statistical tools for online monitoring and analysis.
2. [hexsample](https://github.com/lucabaldini/hexsample):
   event-level analysis for solid-state hybrid detectors with
   hexagonal sampling.
3. [baldaquin](https://github.com/lucabaldini/baldaquin):
   a general-purpose data acquisition framework.
